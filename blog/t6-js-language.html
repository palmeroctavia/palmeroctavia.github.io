Javascript Language

15 Akuhata

In the JavaScript language, describe all the use cases for each of:

parentheses () - e.g. '1. Execute a function var result = doSomething()' - wrapping strings or the conditions of a loop, also use for wrapping names. brackets [] - refering to the position of an item in an array or an array initialiser. braces {} - include the parameters of an object. single quotes '' - generally depends on preference of developer, key is to remain consistent throughout. Typically I favour single quotes and only use double quotes for string content such as speech/quotes. double quotes "" - refer to previous explanation.

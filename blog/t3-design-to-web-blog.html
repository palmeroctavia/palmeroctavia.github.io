What a responsive site is, and why responsiveness is important.

A responsive site is one which adjusts according to the media device being used. This is important as devices are all different sizes and layouts. By making a site responsive it helps ensure the site is still user-friendly no matter whether the user is reading the content off a phone, desktop or tablet.

What mobile first design is, and why it's important.

Mobile first design is an approach to website development which mandates that sites are built for mobile devices first before other layouts are considered. It is important because mobile first design means that sites are configured for the most commonly used viewing device first.

What frameworks are, and their pros and cons?

Frameworks are collections of code that have functionality most developers wish to use such as grid layouts and buttons. Examples of frameworks include Bootstrap and Skeleton. Their main pro is their efficiency (if you want to use certain features without setting up all the styling coding you can). Main cons are that they are often difficult to adjust if you want to style core features like tables and buttons differently and they mean you often need more skill rather than less to successfully use (not so helpful for a beginner like me IMO).
